# Retail Sales Data Analyzer

## Project Overview

Retail Sales Data Analyzer is a Python-based data analysis project developed using Object-Oriented Programming.

The project loads a retail sales dataset, performs data cleaning, calculates business metrics, filters records, and visualizes sales trends using charts.

---

## Features

- Load Retail Sales CSV dataset
- Handle missing values
- Convert Date column
- Display dataset summary
- Calculate total and average sales
- Find most popular product
- Calculate sales growth percentage
- Filter data by Category
- Filter data by Date
- Bar Chart
- Line Graph
- Correlation Heatmap

---
